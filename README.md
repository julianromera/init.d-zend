### [init.d-zend]()


Configurable Init.d script, for Zend Server + Nginx in Debian GNU\Linux

### installation

    # cd /etc/init.d
    # wget https://raw.github.com/julianromerajuarez/init.d-zend/master/zend
    # update-rc.d play defaults
